# Project Brief

## 서비스명
반납지 찾기

## 한 줄 설명
지도를 이동하여 차량 반납 위치를 설정하고, 내비게이션으로 바로 연결할 수 있는 모바일 서비스

## 타겟 사용자
카셰어링 이용자, 렌터카 이용자

## 핵심 기능
1. 지도 기반 반납지 설정 — 지도를 이동하여 원하는 반납 위치를 선택
2. 반납 가능 영역 표시 — 현 위치 반경 1km 영역을 지도에 시각적으로 표시
3. 도로명 주소 표기 — 지도 중심 위치의 도로명 주소를 실시간으로 표시
4. 내비게이션 연결 — 네이버지도, 카카오맵, 티맵으로 바로 길안내 연결
5. 반납 혜택 안내 — 빠른 차량 회수, 주차비 미부담 등 사용자 혜택 표시

## 추가 요구사항
- 모바일 최적화 (모바일 전용 서비스)
- 반납 가능 영역은 현 위치 반경 1km
- 유료 주차장 반납 시 주차비는 사용자 부담 아님
